# Tailwind Mini Projects

There are five simple pages that built with tailwind css in this repo.

## Getting Started

This monorepo project is managed by pnpm, so you must install pnpm first.

```bash
# Install all dependencies
pnpm i
# run dev server
pnpm dev
```
